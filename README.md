# samples-generation

Generating samples of data by database itself.

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

## Building

```bash
pnpm build
```

## Scripts

- `check.sh` - Runs formatting, linting, type checking, and tests
- `health.sh` - Checks for security vulnerabilities and outdated dependencies
- `all-checks.sh` - Runs both check.sh and health.sh
